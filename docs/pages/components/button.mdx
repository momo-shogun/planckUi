## Props table

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | 'primary' \| 'outline' \| 'ghost' | 'primary' | Visual style |
| size | 'sm' \| 'md' \| 'lg' | 'md' | Padding and radius |
| loading | boolean | false | Shows spinner |
| disabled | boolean | false | Disables interaction |
| unstyled | boolean | false | Strips all styles |
| slots.root | ViewStyle | — | Container override |
| slots.text | TextStyle | — | Label override |
| slots.loader | ViewStyle | — | Spinner override |
| onPress | () => void | — | Press handler |

## Slots override example

```tsx
<Button
  variant="primary"
  onPress={() => {}}
  slots={{
    root: { borderRadius: 0 },
    text: { letterSpacing: 2, textTransform: 'uppercase' },
  }}
>
  Custom label
</Button>
```

## Unstyled example

```tsx
import { View, Text } from 'react-native';
import { Button } from '@my-ui-lib/core';

<Button unstyled onPress={() => {}}>
  <View style={{ padding: 12, backgroundColor: '#eee' }}>
    <Text>Custom layout</Text>
  </View>
</Button>
```
