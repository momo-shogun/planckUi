# Button

Planck ships **three button surfaces** from `@my-ui-lib/core`:

1. **`PillButtonV1`** ‚Äî shadcn-style pill (`default` gradient, `secondary`, `outline`, `ghost`), sizes, loading. Also exported as **`Button`** (same implementation).
2. **`ButtonIconOnly`** ‚Äî circular icon actions with variants and optional `active` styling.
3. **`MarqueeButton`** ‚Äî pill with **Reanimated** horizontal scrolling text (native apps; docs use a web mock).

## `PillButtonV1` / `Button` props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | default \| secondary \| outline \| ghost | default | Visual style (`default` uses SVG gradient) |
| size | sm \| md \| lg | md | Height and typography |
| backgroundGradientColors | readonly [string, string] | solid dark | Gradient stops for `default` |
| textColor | string | ‚Äî | Label color override |
| loading | boolean | false | Shows spinner |
| disabled | boolean | false | Disables press |
| unstyled | boolean | false | Raw `Pressable` + children |
| slots | `{ root?, text? }` | ‚Äî | Style overrides |
| onPress | () => void | ‚Äî | Press handler |
| children | ReactNode | ‚Äî | Label (string/number ‚Üí `Text`) |
