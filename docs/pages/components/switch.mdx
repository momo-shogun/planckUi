# Switch

Track + thumb with **opacity layering** and **`Animated.spring`** on the thumb. Optional label.

## Preview

<Preview>
  <Switch label="Notifications" defaultChecked />
  <Switch label="Off" />
</Preview>

## Props

| Prop | Type | Description |
|------|------|-------------|
| checked | boolean | Controlled |
| defaultChecked | boolean | Uncontrolled |
| onChange | (v: boolean) => void | Toggle handler |
| label | string | Optional label |
| size | sm \| md \| lg | Scale |
| unstyled | boolean | No library styles |
| slots | SwitchSlots | Overrides |

## Install

```bash
npx my-ui-lib add switch
```
