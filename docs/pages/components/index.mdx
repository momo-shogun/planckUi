# Components

Library UI you can copy into your app. Pick an item in the **sidebar under Components** to open its docs and live preview.

| Name | Description |
|------|-------------|
| [Avatar](/components/avatar) | Image, initials fallback, status badge |
| [Badge](/components/badge) | Intents, dot mode, sizes |
| [Bottom Sheet](/components/bottom-sheet) | Themed `@gorhom/bottom-sheet` modal — full copy-paste setup on the doc page |
| [Bottom Tabs Navigator](/components/bottom-tabs-navigator) | Plank Bar V1 and V2 bottom tab patterns |
| [Button](/components/button) | PillButtonV1 (`Button`), `ButtonIconOnly`, `MarqueeButton` — previews + source explorer |
| [Cards (MPCard)](/components/cards) | Profile / match card with gradient and icon actions |
| [Checkbox](/components/checkbox) | Animated check / indeterminate |
| [Dropdown & MultiSelect](/components/dropdown-menu) | Single / multi select, search, anchored or modal |
| [Header](/components/header) | App header variants (PlanckH1V1) |
| [Input](/components/input) | Label, placeholder, error state |
| [Modal](/components/modal) | Portal dialog, header/body/footer |
| [Navigation](/components/navigation/drawer-content) | Drawer list, tab bar, header, back |
| [Pressable](/components/pressable) | Low-level press target |
| [Stack](/components/stack) | HStack / VStack layout |
| [Switch](/components/switch) | Animated thumb, sizes |
| [Tabs](/components/tabs) | Underline / pill / bordered + scrollable; **ZeptoTabs** (Reanimated) on a separate import |
| [Text](/components/text) | Typography variants |
| [Toast](/components/toast) | Provider, queue, portal |
