# Text

**Text** is the typography primitive. It reads font size, weight, and default color from the active theme via the **`variant`** prop.

## Variants

| variant | Default size | Default weight | Default color |
|---------|--------------|----------------|---------------|
| `heading` | `theme.fontSizes.xl` | bold | `theme.colors.textPrimary` |
| `body` | `theme.fontSizes.md` | regular | `theme.colors.textPrimary` |
| `label` | `theme.fontSizes.sm` | medium | `theme.colors.textSecondary` |
| `caption` | `theme.fontSizes.xs` | regular | `theme.colors.textDisabled` |

## Overrides

Use **`color`** for any CSS color string and **`weight`** for `fontWeight` overrides:

```tsx
<Text variant="body" color="#7c3aed" weight="700">
  Emphasized body
</Text>
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| variant | `'heading'` \| `'body'` \| `'label'` \| `'caption'` | `'body'` | Preset typography |
| color | string | — | Overrides variant text color |
| weight | TextStyle['fontWeight'] | — | Overrides variant font weight |

Additional `Text` props from React Native (`numberOfLines`, `onPress`, etc.) are forwarded.
