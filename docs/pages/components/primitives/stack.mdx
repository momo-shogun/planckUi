# Stack

The **Stack** primitive lays out children in a flex container. **HStack** is a row (`direction="row"`), **VStack** is a column (`direction="column"`).

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| direction | `'row'` \| `'column'` | `'column'` | Flex direction (`Stack` only) |
| gap | number | 0 | Gap between children (flex gap) |
| align | Flex `alignItems` | — | Cross-axis alignment |
| justify | Flex `justifyContent` | — | Main-axis justification |

`HStack` / `VStack` accept the same props as `Stack` except `direction` is fixed.

## HStack — row with gap

```tsx
import { HStack } from '@my-ui-lib/core';
import { View } from 'react-native';

<HStack gap={12} align="center">
  <View style={{ width: 40, height: 40, backgroundColor: '#3b82f6' }} />
  <View style={{ width: 40, height: 40, backgroundColor: '#22c55e' }} />
  <View style={{ width: 40, height: 40, backgroundColor: '#eab308' }} />
</HStack>
```

## VStack — column

```tsx
import { VStack } from '@my-ui-lib/core';
import { View } from 'react-native';

<VStack gap={8}>
  <View style={{ height: 24, backgroundColor: '#94a3b8' }} />
  <View style={{ height: 24, backgroundColor: '#64748b' }} />
</VStack>
```

## Nested layout

```tsx
<VStack gap={16} align="stretch">
  <HStack gap={8} justify="space-between">
    <View style={{ flex: 1, height: 32, backgroundColor: '#cbd5e1' }} />
    <View style={{ width: 32, height: 32, backgroundColor: '#64748b' }} />
  </HStack>
  <View style={{ height: 40, backgroundColor: '#e2e8f0' }} />
</VStack>
```
