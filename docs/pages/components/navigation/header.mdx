# Header

**`Header`** supports **`default`**, **`transparent`**, and **`blurred`** (semi-opaque surface via tokens — no extra native blur dependency in core). Pass **`left`** / **`right`** slots for actions.

## Stack example

```tsx
import { createNativeStackNavigator } from '@react-navigation/native-stack';
import { Pressable, Text } from '@my-ui-lib/core';

<Stack.Screen
  name="Detail"
  options={({ navigation }) => ({
    headerShown: true,
    header: () => (
      <Header
        title="Detail"
        left={
          <Pressable accessibilityLabel="Back" onPress={() => navigation.goBack()}>
            <Text>Back</Text>
          </Pressable>
        }
      />
    ),
  })}
/>
```

## Install

```bash
npx my-ui-lib add header
```
