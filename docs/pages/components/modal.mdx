# Modal

Themed dialog rendered through **`@gorhom/portal`**, with backdrop + surface **`Animated`** scale/opacity. Use **`visible`**, **`onClose`**, and optional **`ModalHeader`**, **`ModalBody`**, **`ModalFooter`**. Hardware back (Android) and backdrop tap can dismiss when enabled.

Wrap your app with **`PortalProvider`** (re-exported from `@my-ui-lib/core`) above screens that open modals.

## Props

| Prop | Type | Description |
|------|------|-------------|
| visible | boolean | Shows or hides the modal |
| onClose | () => void | Called on dismiss |
| closeOnBackdrop | boolean | Backdrop tap closes (default true) |
| closeOnBack | boolean | Android back closes (default true) |
| size | 'sm' \| 'md' \| 'lg' \| 'full' | Max width / layout |
| unstyled | boolean | Raw React Native `Modal` only |
| slots | object | `overlay`, `backdrop`, `surface`, `content` |

## Animation

Entrance uses parallel timing + spring; **reduced motion** (via `AccessibilityInfo`) snaps to the final state.

## Install

```bash
npx my-ui-lib add modal
```
