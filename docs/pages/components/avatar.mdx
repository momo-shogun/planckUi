# Avatar

Image with **fallback** initials, optional **badge** (online / offline / busy). Colors come from **`getAvatarTokens`** only.

## Preview

<Preview>
  <Avatar fallback="Ada Lovelace" size="md" />
  <Avatar fallback="X" size="sm" badge="online" />
</Preview>

## Props

| Prop | Type | Description |
|------|------|-------------|
| src | string | Image URI |
| fallback | string | Initials / label when no image |
| size | xs … xl | Diameter scale |
| variant | circle \| rounded \| square | Shape |
| badge | online \| offline \| busy | Status dot |
| unstyled | boolean | No library styles |
| slots | AvatarSlots | Overrides |

## Install

```bash
npx my-ui-lib add avatar
```
