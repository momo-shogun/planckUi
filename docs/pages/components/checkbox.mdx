# Checkbox

Controlled checkbox with **Animated** check / indeterminate, a11y **`checkbox`**, and sizes.

## Preview

<Preview>
  <Checkbox label="Remember me" defaultChecked />
  <Checkbox label="Indeterminate" indeterminate />
</Preview>

## Props

| Prop | Type | Description |
|------|------|-------------|
| checked | boolean | Controlled |
| defaultChecked | boolean | Uncontrolled default |
| onChange | (v: boolean) => void | Change handler |
| indeterminate | boolean | Indeterminate bar |
| label | string | Label text |
| size | sm \| md \| lg | Scale |
| unstyled | boolean | No library styles |
| slots | CheckboxSlots | Overrides |

## Animation

Respects **reduced motion** via shared `useReducedMotionRef` pattern inside the component.

## Install

```bash
npx my-ui-lib add checkbox
```
