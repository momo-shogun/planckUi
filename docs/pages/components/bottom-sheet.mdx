# Bottom Sheet

Themed **`BottomSheet`** wraps **`BottomSheetModal`** from [`@gorhom/bottom-sheet`](https://gorhom.dev/react-native-bottom-sheet/). It uses design tokens for the sheet surface, handle, title, and content area. **`BottomSheetProvider`** is a re-export of **`BottomSheetModalProvider`** — mount it once near the root of your app.

<Callout type="info">
  In the **planckUi example** app, open **Sheet lab** from the drawer to try the same pattern live.
</Callout>

## Requirements

Your app must use **`react-native-reanimated`** (with the **Reanimated Babel plugin last** in `babel.config.js`), **`react-native-gesture-handler`**, and wrap the UI in **`GestureHandlerRootView`**. See the [Reanimated](https://docs.swmansion.com/react-native-reanimated/) and [Gesture Handler](https://docs.swmansion.com/react-native-gesture-handler/docs/installation) install guides.

```js filename="babel.config.js"
module.exports = {
  presets: ['module:@react-native/babel-preset'],
  plugins: [
    // …other plugins
    'react-native-reanimated/plugin', // must be last
  ],
};
```

## Install

```bash
npx my-ui-lib add bottom-sheet
```

Or ensure the library and Gorhom packages are installed in your app (versions should match your React Native release):

```bash
pnpm add @my-ui-lib/core @my-ui-lib/tokens @gorhom/bottom-sheet react-native-reanimated react-native-gesture-handler
```

## Setup + usage (copy & paste)

Minimal **`App.tsx`** pattern: provider + theme + one screen that toggles **`visible`**.

```tsx filename="App.tsx"
import React, { useState } from 'react';
import { View } from 'react-native';
import { GestureHandlerRootView } from 'react-native-gesture-handler';
import {
  BottomSheet,
  BottomSheetProvider,
  Button,
  Text,
  ThemeProvider,
  useTheme,
} from '@my-ui-lib/core';
import { defaultTheme } from '@my-ui-lib/tokens';

export default function App() {
  return (
    <GestureHandlerRootView style={{ flex: 1 }}>
      <BottomSheetProvider>
        <ThemeProvider theme={defaultTheme}>
          <SheetDemoScreen />
        </ThemeProvider>
      </BottomSheetProvider>
    </GestureHandlerRootView>
  );
}

function SheetDemoScreen() {
  const theme = useTheme();
  const [open, setOpen] = useState(false);

  return (
    <View
      style={{
        flex: 1,
        padding: theme.spacing[4],
        backgroundColor: theme.colors.background,
      }}>
      <Text variant="heading" style={{ marginBottom: theme.spacing[3] }}>
        Bottom sheet
      </Text>
      <Button onPress={() => setOpen(true)}>Open sheet</Button>
      <BottomSheet
        visible={open}
        onClose={() => setOpen(false)}
        title="Sheet title"
        snapPoints={['40%', '70%']}>
        <Text color={theme.colors.textSecondary}>
          Put any React Native views here. Drag the handle or backdrop to dismiss
          (same behavior as @gorhom/bottom-sheet).
        </Text>
      </BottomSheet>
    </View>
  );
}
```

If you already use **`PortalProvider`** for **`Modal`**, keep the same order as the example app: **`BottomSheetProvider`** → **`PortalProvider`** → the rest of the tree (both can wrap **`ThemeProvider`** or sit outside it, as long as **`ThemeProvider`** wraps any screen that renders **`BottomSheet`** so tokens resolve).

```tsx
<BottomSheetProvider>
  <PortalProvider>
    <ThemeProvider theme={defaultTheme}>{/* … */}</ThemeProvider>
  </PortalProvider>
</BottomSheetProvider>
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| visible | boolean | Present / dismiss |
| onClose | () => void | Maps to `onDismiss` |
| snapPoints | (string \| number)[] | Default `['45%','80%']` |
| title | string | Optional heading |
| modalProps | object | Extra `BottomSheetModal` props |
| unstyled | boolean | Skip theme surface styles |
| slots | BottomSheetSlots | `sheet`, `content` |

## Imperative ref (optional)

The underlying ref type is exported as **`BottomSheetModalRef`** from **`@my-ui-lib/core`** if you need **`present` / `dismiss`** outside of **`visible`**. Most apps only need **`visible`** + **`onClose`**.
