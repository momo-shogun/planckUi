# Toast

**`ToastProvider`** + **`useToast`** queue up to **three** toasts, rendered through **`@gorhom/portal`**. Each toast animates a **progress** bar with **`Animated.timing`** (disabled path for reduced motion uses a simple timeout).

Place **`ToastProvider`** inside **`ThemeProvider`** (tokens resolve in the provider). Include **`PortalProvider`** at the app root.

## API

| Export | Description |
|--------|-------------|
| `showToast(options)` | `{ title, description?, intent?, duration?, id? }` → toast id |
| `dismissToast(id)` | Remove by id |

## Install

```bash
npx my-ui-lib add toast
```
